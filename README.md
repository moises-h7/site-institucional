# Bookshelf - Site Institucional

Este é o projeto desenvolvido para o **Estudo Dirigido 1: Criação de Site Institucional**. O objetivo é criar um site de apresentação para uma empresa fictícia utilizando as tecnologias fundamentais da web: HTML5 e CSS3.

## 📚 Sobre o Projeto

O **Bookshelf** é uma biblioteca local fictícia dedicada à leitura, ao conhecimento e ao fortalecimento da comunidade. O site foi estruturado para ser simples, semântico e visualmente agradável, com foco na experiência do usuário.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo (`<header>`, `<main>`, `<nav>`, `<footer>`, `<section>`).
- **CSS3**: Estilização visual, layout responsivo e design moderno.

## 📂 Estrutura do Projeto

```text
projeto-ed1/
├── index.html        # Página Inicial (Home)
├── sobre.html        # Página Sobre nós
├── contato.html      # Página de Contato
├── style.css         # Arquivo de estilos (CSS)
└── img/              # Pasta de imagens e logotipos
```

## ✨ Funcionalidades

1.  **Home**: Apresentação geral da biblioteca e destaques de acervo.
2.  **Sobre**: História da fundação, missão, visão, valores e apresentação da equipe.
3.  **Contato**: 
    - Formulário funcional com campos para Nome, E-mail e Mensagem.
    - Redirecionamento automático para a página inicial ao enviar.
4.  **Design Responsivo**: Layout que se adapta a diferentes tamanhos de tela (desktop e mobile).
5.  **Menu de Navegação**: Navegação fluida entre todas as páginas do site.

## 🛠️ Como visualizar

Para abrir o projeto localmente:

1. Faça o clone ou download deste repositório.
2. Navegue até a pasta `projeto-ed1`.
3. Abra o arquivo `index.html` em qualquer navegador web (Chrome, Firefox, Edge, Safari).
